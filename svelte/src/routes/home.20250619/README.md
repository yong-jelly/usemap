# 홈 피드 시스템 v2.0

## 🎯 개요

사용자가 직접 입력하지 않고 자동으로 큐레이션되는 SNS 스타일의 맞춤형 피드 시스템입니다. 음식점, 유튜브, 통계, 게이미피케이션 등 다양한 콘텐츠를 카드 형태로 제공합니다.

## 🚀 주요 기능

### 📱 피드 헤더 (`FeedHeader.svelte`)
- **사용자 상태 위젯**: 레벨, 경험치, 방문 통계, 연속 활동일
- **탭 네비게이션**: 전체, 추천, 트렌딩, 통계, 도전
- **실시간 알림**: 새로운 도전 과제, 추천 알림
- **설정**: 개인화 설정 및 알림 관리

### 🎴 카드 컴포넌트들

#### 1. 음식점 추천 카드 (`RestaurantRecommendCard.svelte`)
- **AI/알고리즘 추천**: 개인 취향 기반 맞춤 추천
- **추천 이유**: 상세한 AI 분석 및 설명
- **매치율**: 개인 취향과의 일치도 (%)
- **음식점 정보**: 평점, 리뷰 수, 카테고리, 가격대
- **태그**: 관련 키워드 및 특징

#### 2. 유튜브 추천 카드 (`YoutubeRecommendCard.svelte`)
- **영상 썸네일**: 클릭 시 유튜브로 이동
- **관련 음식점**: 영상에서 소개된 맛집 목록
- **요약**: AI 생성 영상 요약
- **채널 정보**: 구독자, 조회수, 업로드 날짜

#### 3. 통계 카드 (`StatsCard.svelte`)
- **개인 통계**: 방문, 좋아요, 저장 활동 요약
- **순위 현황**: 지역별/전국 순위
- **달성 현황**: 뱃지 및 성과 관리
- **증감률**: 이전 기간 대비 활동 변화

#### 4. 도전 과제 카드 (`ChallengeCard.svelte`)
- **게이미피케이션**: 일일/주간/월간 도전 과제
- **진행률**: 실시간 진행 상황 표시
- **보상 시스템**: EXP, 뱃지, 특별 혜택
- **난이도**: 쉬움/보통/어려움 구분

#### 5. 내 활동 통계 카드 (`MyActivityStatsCard.svelte`)
- **7일간 활동 요약**: 좋아요, 방문, 저장, 추천 등록, 리뷰 작성, 조회 수
- **증감률 표시**: 이전 주 대비 활동 변화율
- **활동 카테고리**: 상호작용, 콘텐츠 생성, 탐색 활동으로 그룹화
- **상세 통계**: 클릭 시 개별 활동 상세 보기

#### 6. 내 추천 음식점 목록 (`MyRestaurantList.svelte`)
- **음식점 목록**: 내가 추천한 음식점들의 상세 통계
- **개별 통계**: 조회수, 좋아요, 저장, 방문 수 (7일간/전체)
- **참여율**: 조회 대비 상호작용 비율
- **내 기여도**: 추천 수, 리뷰 수 표시

#### 7. 내 취향 분석 차트 (`MyPreferencesChart.svelte`)
- **지역별 선호도**: 방문/상호작용 기반 지역 선호도 차트
- **카테고리별 선호도**: 음식 종류별 선호도 분석
- **다양성 점수**: 취향의 다양성 수치화 (0-100점)
- **취향 분석 결과**: AI 기반 개인 취향 요약

## 🎨 UI/UX 특징

### 디자인 시스템
- **모던한 카드 디자인**: 그림자, 둥근 모서리, 깔끔한 레이아웃
- **컬러 시스템**: 카테고리별 고유 색상 (음식점-파랑, 유튜브-빨강, 통계-보라, 도전-인디고)
- **반응형**: 모바일 퍼스트 디자인
- **다크모드**: 지원 예정

### 인터랙션
- **무한 스크롤**: 자동 콘텐츠 로딩
- **Pull-to-refresh**: 새 콘텐츠 업데이트
- **탭 기반 필터링**: 원하는 콘텐츠만 선별 조회
- **실시간 알림**: 새로운 콘텐츠 및 도전 과제 알림

```
│       ├── StatsCard.svelte

```

### 데이터 모델

#### FeedItem 인터페이스
```typescript
interface FeedItem {
  id: string;
  type: 'restaurant' | 'youtube' | 'stats' | 'challenge' | 'community' | 'my_activity' | 'my_restaurants' | 'my_preferences';
  timestamp: string;
  data: any; // 각 카드 타입별 데이터
}
```

#### 각 카드별 데이터 구조
- **RestaurantRecommendation**: 음식점 정보 + 추천 알고리즘 결과
- **YoutubeRecommendation**: 영상 정보 + 관련 음식점 + AI 요약
- **StatsData**: 사용자 통계 + 순위 + 달성 정보
- **Challenge**: 도전 과제 + 진행률 + 보상 정보

## 🤖 AI/알고리즘 기능

### 추천 시스템
1. **개인화 추천**: 사용자 행동 패턴 분석
2. **협업 필터링**: 비슷한 취향 사용자 기반 추천
3. **콘텐츠 기반**: 음식점 특성 및 리뷰 분석
4. **위치 기반**: GPS 위치 정보 활용

### 자동 큐레이션
- **PostgreSQL cron**: 정기적 데이터 집계
- **실시간 트렌딩**: 인기 급상승 콘텐츠 감지
- **계절성 반영**: 시기별 맞춤 추천

## 🎮 게이미피케이션

### 레벨 시스템
- **경험치(EXP)**: 활동 기반 포인트 획득
- **레벨업**: 새로운 기능 및 혜택 해제
- **뱃지**: 특별한 성과 인증

### 도전 과제
- **일일**: 새로운 맛집 1곳 방문하기
- **주간**: 5곳 이상 음식점 방문하기
- **월간**: 10개 카테고리 음식 체험하기
- **특별**: 제주도 맛집 탐방하기

### 순위 시스템
- **지역별 순위**: 동네 맛집 탐험왕
- **카테고리별**: 한식 전문가, 카페 마니아 등
- **전국 순위**: 대한민국 맛집 탐험가

## 📊 통계 및 분석

### 개인 통계
- **활동 요약**: 방문, 좋아요, 저장, 리뷰 작성 수
- **성장 지표**: 이전 기간 대비 증감률
- **취향 분석**: 선호 음식 카테고리, 가격대, 지역

### 전체 통계
- **트렌딩**: 실시간 인기 음식점 및 지역
- **계절 트렌드**: 시기별 인기 메뉴 및 장소
- **지역 특색**: 각 지역별 특징적인 맛집
