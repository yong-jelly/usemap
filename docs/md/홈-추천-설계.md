# 홈 추천 탭 (/home?view=recommend) 설계

## 1. 활동중인 맛탐정 (Active Detectives)

### 1.1 개요
- 최근 1개월간 자신의 폴더에 음식점을 추가한 이력이 있는 활발한 사용자들을 소개하는 섹션입니다.
- 사용자별로 최근 추가한 음식점 수(뱃지)를 표시하여 활동성을 강조합니다.

### 1.2 데이터 요구사항 (DB/API)
- **대상**: 최근 30일 이내 `tbl_folder_place`에 레코드를 추가한 사용자
- **집계**: 사용자별 추가한 음식점 수 (중복 제거 여부 검토 필요 - 동일 음식점을 여러 폴더에 넣었을 경우)
- **정렬**: 최근 활동 순 또는 활동량 순
- **필요 데이터**:
    - `user_id`, `nickname`, `profile_image`
    - 최근 1개월간 추가한 음식점 수 (`count`)
    - 최근 추가한 음식점들의 썸네일 이미지 목록 (가로 슬라이드용)

### 1.3 UI/UX 구성
- **홈 섹션**:
    - 타이틀: "활동중인 맛탐정"
    - 우측 상단: "전체 보기" 링크
    - 리스트: 가로 스크롤 형태의 사용자 카드
        - 사용자 프로필 이미지 + 닉네임
        - 활동 뱃지: "N개 추가" (최근 1개월 기준)
- **'전체 보기' 페이지 (/home/active-detectives)**:
    - 최근 활동 사용자 TOP 100 목록
    - 각 항목 구성:
        - 좌측: 사용자 프로필 (이미지 클릭 시 `/p/user/{user_id}` 이동)
        - 우측: 최근 추가한 음식점 썸네일 목록 (가로 슬라이드 컴포넌트 사용)
    - 클릭 시 동작:
        - 사용자 프로필 이미지 클릭: 해당 사용자의 프로필 페이지로 이동
        - 그 외 영역 클릭: 해당 사용자가 폴더에 등록한 전체 음식점 목록 페이지로 이동

### 1.4 유저별 등록 음식점 목록 페이지
- **경로**: `/profile/active-places/{user_id}` (또는 유사한 경로)
- **스타일**: `/profile/liked`와 유사한 그리드/리스트 뷰
- **데이터 로직**:
    - 해당 유저가 `tbl_folder_place`에 등록한 모든 음식점
    - **중복 제거**: 여러 폴더에 담긴 동일 음식점은 하나만 표시
    - **정렬**: `created_at` 내림차순 (최근 등록순)
