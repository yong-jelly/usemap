# 유지보수 쿼리

`tbl_place` 등 장소 관련 테이블의 데이터 정리·검증용 쿼리 모음임.

## 1. 음식점 판별 기준 (참고)

- **음식점**: `category_code_list[1] = '220036'`
- **예외 포함**: `category = '떡,한과'` (루트 223563이지만 음식 판매점으로 포함)
- **비음식점**: 위 조건에 해당하지 않는 데이터

## 2. 데이터 정리 쿼리

### 2.1 이미지 없는 장소 삭제

`images`와 `place_images`가 모두 비어 있거나 `NULL`인 데이터 정리함.

```sql
-- 대상 개수 확인
SELECT count(*) FROM public.tbl_place 
WHERE (images IS NULL OR array_length(images, 1) IS NULL)
  AND (place_images IS NULL OR array_length(place_images, 1) IS NULL);

-- 삭제 (FK 참조 시 아래 2.3 순서 참고)
DELETE FROM public.tbl_place 
WHERE (images IS NULL OR array_length(images, 1) IS NULL)
  AND (place_images IS NULL OR array_length(place_images, 1) IS NULL);
```

### 2.2 비음식점 삭제 전 확인

```sql
SELECT COUNT(*) FROM tbl_place 
WHERE (category_code_list[1] != '220036' OR category_code_list IS NULL)
  AND category != '떡,한과';
```

### 2.3 비음식점 데이터 제거 (연관 테이블 포함, 순서 중요)

`tbl_folder_place` 등이 `tbl_place`를 FK로 참조하므로, 아래 순서대로 실행해야 함.

```sql
-- 1. tbl_folder_review
DELETE FROM tbl_folder_review 
WHERE place_id IN (
  SELECT id FROM tbl_place 
  WHERE (category_code_list[1] != '220036' OR category_code_list IS NULL)
    AND category != '떡,한과'
);

-- 2. tbl_folder_place
DELETE FROM tbl_folder_place 
WHERE place_id IN (
  SELECT id FROM tbl_place 
  WHERE (category_code_list[1] != '220036' OR category_code_list IS NULL)
    AND category != '떡,한과'
);

-- 3. tbl_naver_folder_place
DELETE FROM tbl_naver_folder_place 
WHERE place_id IN (
  SELECT id FROM tbl_place 
  WHERE (category_code_list[1] != '220036' OR category_code_list IS NULL)
    AND category != '떡,한과'
);

-- 4. tbl_instagram_place
DELETE FROM tbl_instagram_place 
WHERE place_id IN (
  SELECT id FROM tbl_place 
  WHERE (category_code_list[1] != '220036' OR category_code_list IS NULL)
    AND category != '떡,한과'
);

-- 5. tbl_place
DELETE FROM tbl_place 
WHERE (category_code_list[1] != '220036' OR category_code_list IS NULL)
  AND category != '떡,한과';
```

## 3. 검증·집계 쿼리

### 3.1 음식점 vs 비음식점 비율

```sql
SELECT 
  CASE WHEN category_code_list[1] = '220036' THEN '음식점' ELSE '비음식점' END AS type,
  COUNT(*) AS cnt,
  ROUND(COUNT(*) * 100.0 / SUM(COUNT(*)) OVER(), 2) AS pct
FROM tbl_place
GROUP BY 1;
```

### 3.2 테이블별 레코드 수 현황

```sql
SELECT 'tbl_place' AS table_name, COUNT(*) AS cnt FROM tbl_place
UNION ALL SELECT 'tbl_place (음식점)', COUNT(*) FROM tbl_place WHERE category_code_list[1] = '220036'
UNION ALL SELECT 'tbl_naver_folder', COUNT(*) FROM tbl_naver_folder
UNION ALL SELECT 'tbl_naver_folder_place', COUNT(*) FROM tbl_naver_folder_place
UNION ALL SELECT 'tbl_place_features', COUNT(*) FROM tbl_place_features
UNION ALL SELECT 'tbl_folder_place', COUNT(*) FROM tbl_folder_place;
```

### 3.3 루트 카테고리별 장소 수

```sql
SELECT category_code_list[1] AS root_code, COUNT(*) AS cnt
FROM tbl_place 
WHERE category_code_list IS NOT NULL
GROUP BY 1 ORDER BY 2 DESC;
```

### 3.4 비음식점 카테고리 분포 (삭제 대상 확인용)

```sql
SELECT category, category_code_list[1] AS root_code, COUNT(*) AS cnt
FROM tbl_place 
WHERE category_code_list[1] != '220036' OR category_code_list IS NULL
GROUP BY 1, 2 ORDER BY 3 DESC LIMIT 20;
```
