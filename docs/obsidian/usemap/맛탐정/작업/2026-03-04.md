# 2026-03-04 작업 정리

## 개요

장소 상세 모달에 "포함된 콜렉션" 섹션 추가. v1_get_place_features API 재설계 및 UI 구현.

## 1. v1_get_place_features API 재설계

### 변경 내용

- 기존 컬럼 4개(recent_thumbnails, group_place_count, group_subscriber_count, group_key)를 `collection` jsonb 하나로 통합함.
- SQL 파일: `docs/sql/101_update_v1_get_place_features_collection.sql`

### collection 구조

```json
{
  "key": "그룹 식별자 (프론트 중복 제거용)",
  "name": "콜렉션(채널) 표시명",
  "places": [
    { "url": "썸네일 URL", "place_name": "장소명" }
  ]
}
```

- `places`: 최근 추가된 음식점 최대 3개 (최신순), 각 항목에 썸네일 포함함.

### platform_type별 collection 구성

| platform_type | key | name | places 소스 |
|---------------|-----|------|-------------|
| youtube | channelId | channelTitle | 같은 채널 최신 영상 썸네일 3개 |
| folder | folder_id | 폴더명 | 폴더 내 장소 images[1] 3개 |
| community | domain | domain | 같은 도메인 장소 이미지 3개 |
| public_user | folder.id | 폴더명 | 폴더 내 장소 images[1] 3개 |
| social | service | service | 같은 서비스 장소 이미지 3개 |

## 3. PlaceIncludedCollections UI

**파일**: `src/features/place/ui/PlaceIncludedCollections.tsx`

- 장소가 포함된 콜렉션을 가로 스크롤 카드 형태로 표시함.
- `collection.key` 기준 중복 제거함.
- 썸네일: `collection.places[0]?.url` 사용, 없으면 장소 이미지 fallback 또는 placeholder 아이콘.
- 썸네일 하단에 콜렉션(채널) 이름만 표시함. 카운트 정보 없음.
- 커뮤니티 도메인 한글 변환: 클리앙, 다모앙, 보배드림 등.
