# 맛탐정 주요 기능 (개념)

## 1. 맛탐정(폴더) 시스템

- 사용자가 장소를 큐레이션·관리하는 폴더 기능
- 플레이리스트와 유사한 개념
- 장소 추가/삭제, 다른 사용자 폴더 구독, 피드로 새 소식 확인

## 2. 구독 피드

- 다양한 소스(네이버 지도, 유튜브, 인스타그램 등) 콘텐츠 구독
- 피드 형식으로 확인
- 최신순/거리순 정렬, 위치 기반 필터링

## 3. 장소 탐색

- 지역별, 테마별, 카테고리별 장소 검색·탐색
- 여행 시 특정 지역 저장 맛집을 한 번에 확인

## 4. 리뷰 및 방문 기록

- **리뷰·별점**: 기능 없음 또는 소극적 제공. 맛탐정의 핵심은 출처 기반 발견이지 리뷰 기반 추천이 아님.
- 방문 기록 관리, 방문 여부·횟수 추적
- 네이버 별점·리뷰는 네이버 지도 버튼으로 외부 확인 유도

## 5. 지도 기반 탐색

- Mapbox GL 활용
- 지도 기반 장소 탐색
- 폴더 내 맛집 지도 클러스터링 표시

## 6. 소스 통합 현황

| 소스 | 상태 | 설명 |
|------|------|------|
| 유튜브 | 구현됨 | 맛집 채널별 장소 정보 수집 |
| 인스타그램 | 구현됨 | 콘텐츠에서 장소 매핑 |
| 커뮤니티 | 구현됨 | 클리앙, 다모앙, 보배드림 수집 |
| 네이버 플레이스 | 구현됨 | 장소 검색, 상세 조회 |
| 맛탐정 폴더 | 구현됨 | 사용자 공개 큐레이션 |
