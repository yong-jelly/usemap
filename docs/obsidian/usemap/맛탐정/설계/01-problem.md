# 맛탐정이 해결하려는 문제

## 1. 저장은 쉽지만, 관리는 불가능

네이버 지도 별표시, 폴더 저장, SNS 북마크 등으로 수백~천 개의 맛집 목록이 쌓이나, 필요할 때 활용하기 어려움.

- **출처 소실**: 누가, 언제, 어디서 추천했는지 기억나지 않음
- **저장 이유 망각**: 왜 저장했는지 맥락이 사라짐
- **재탐색 불가**: 여행 시 특정 지역 저장 목록을 한 번에 확인 어려움
- **결국 재검색**: 저장 목록 대신 가까운 곳을 새로 검색하게 됨

## 2. 네이버 지도 리뷰·별점의 신뢰도 문제

- 광고 음식점이 상위 노출됨
- 이벤트/마케팅 업체 리뷰 비중이 커 신뢰도 낮음
- 진짜 맛집을 찾으려면 커뮤니티, 유튜브, SNS를 따로 뒤져야 함

## 3. 협업과 공유의 한계

- 네이버 폴더 공유 기능 제한적
- 회사 회식, 친구들과 공동 맛집 목록 관리 어려움
- 폴더 구독 개념 부재로 업데이트 실시간 확인 어려움
- 네이버 계정에 종속되어 장기 사용에 불안정함

## 4. 유튜브 맛집 정보의 접근성 문제

유튜브는 음식점 분위기와 음식을 생생하게 전달하나, 현실적 한계가 있음.

- 인기 먹방 채널 영상 수백~수천 개
- 과거 영상 속 음식점을 하나씩 찾아보기 현실적으로 불가능
- "여기 가고 싶다"는 생각을 따로 메모하기 번거로움
- 유튜브 맛집 정보가 휘발되어 사라짐

> "성시경이 추천한 음식점 어디였지?", "먹방 유튜버가 갔던 그 국밥집이 어디더라?"

## 5. 맥락 소실(Context Loss)

- **맥락 소실**: 기존 저장 방식은 '장소'는 남지만 '왜, 어디서' 저장했는지 기억이 사라짐
- **막연한 나열**: 단순 이미지 위주 구성으로 저장 의도를 즉각 떠올리기 어려움
