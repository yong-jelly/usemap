# 맛탐정(usemap) 프로젝트 개요

## 1. 한 줄 정의

> 흩어진 맛집 정보를 한곳에 모아 체계적으로 관리하고, 신뢰할 수 있는 소스에서 맛집을 발견하는 서비스

## 2. 프로젝트 목적

- 맛집 큐레이션 및 공유 플랫폼
- 사용자 자신만의 맛집 리스트를 폴더 형태로 관리
- 다른 사용자·소스의 큐레이션을 구독하여 피드로 소식 수신

## 3. 핵심 가치 제안

> **"흩어진 맛집 정보의 허브, 그리고 교차 검증 플랫폼"**

> **"리뷰/별점 없음. 출처 정보에 의해 관심·흥미를 가질 음식점을 발견·탐색을 위해 나열함."** — [[07-engagement-flow|상세]]

### 3.1 플랫폼이 제공하는 가치

1. **소스 데이터베이스** (관리자 구축)
   - 유튜브/인스타그램 인플루언서의 맛집 정보 DB화
   - 커뮤니티 맛집 추천글 자동 수집
   - 한 번 구축되면 모든 사용자가 활용하는 공공재

2. **맛탐정 생태계** (사용자 참여)
   - 누구나 자신만의 맛집 큐레이션을 공개하고 구독자 확보 가능
   - 구독자 수와 피드백으로 신뢰도·전문성 축적

3. **교차 검증 시스템** (신뢰도 확보)
   - 하나의 음식점에 여러 소스 연결
   - 맛탐정 + 유튜브 + 커뮤니티 = 광고 리뷰와 다른 객관성
   - 유튜브 링크로 텍스트만으로는 알 수 없는 생생한 정보 제공

## 4. 프로젝트 구조 (개념 수준)

```text
+------------------+     +------------------+     +------------------+
|   사용자(소비자)   | --> |   맛탐정 플랫폼   | <-- |  소스(유튜브 등)  |
+------------------+     +------------------+     +------------------+
        |                        |                        |
        v                        v                        v
   폴더 구독/관리           교차 검증 DB            인플루언서 콘텐츠
   피드 확인               맛탐정 큐레이션         자동 수집
```
