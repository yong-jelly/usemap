# 맛탐정의 해결 전략

## 1. 신뢰할 수 있는 추천 소스 통합

광고·마케팅이 많은 네이버 리뷰 대신, 실제 사용자들이 추천하는 소스들을 통합함.

| 소스 타입 | 설명 | 예시 |
|-----------|------|------|
| 유튜브 | 맛집 인플루언서 영상 속 음식점 DB화 | 성시경, 먹방 채널 등 |
| 인스타그램 | 지역 맛집 인플루언서 게시물 속 음식점 DB화 | 지역 맛집 계정 등 |
| 커뮤니티 | 검증된 맛집 추천 | 클리앙, 다모앙, 보배드림 |
| 네이버 플레이스 폴더 | 테마별 큐레이션 | 테마별 맛집 리스트 |
| 맛탐정 폴더 | 일반 사용자 공개 큐레이션 | 동네 맛집, 회식 장소 등 |

**핵심**: 유튜브·인스타그램 등 외부 소스는 플랫폼 관리자가 지속 추적하며 축적함. 한 번 구축된 DB는 모든 사용자가 활용하는 공공재가 됨.

## 2. 맛탐정: 누구나 큐레이터가 됨

**"맛탐정"은 특별한 사람이 아님. 일반 사용자도 될 수 있음.**

공개 폴더를 만들고 자신만의 맛집 큐레이션을 시작하면 그 순간부터 맛탐정이 됨.

```text
맛탐정 @미식가김철수
├── 결혼기념일 추천 레스토랑 (구독자 234명)
├── 부모님과 함께하기 좋은 곳 (구독자 156명)
├── 아이들과 가기 좋은 식당 (구독자 89명)
└── 혼밥하기 좋은 곳 (구독자 412명)
```

### 맛탐정이 되면 얻는 것

1. **구독자 확보**: 취향이 맞는 사용자들이 폴더 구독
2. **신뢰도 축적**: 꾸준한 추천으로 구독자 증가, 영향력 형성
3. **전문성 인정**: "이 사람 추천이면 믿을 수 있다"는 평판
4. **커뮤니티 기여**: 미식 경험이 다른 사람에게 도움

## 3. 폴더 기반 체계적 관리

- **메모 기능**: 저장 이유, 추천 포인트, 개인 노트 추가
- **출처 추적**: 어디서 발견했는지 자동 기록
- **권한 관리**: 공개, 비공개, 초대 코드 모드
- **협업 편집**: 폴더를 함께 편집하는 참여형 폴더

## 4. 소스 등록과 교차 검증

**핵심은 "소스(출처)"임.**

사용자는 음식점 상세에서 관련 콘텐츠를 직접 등록 가능:
- 유튜브 영상 링크
- 인스타그램/쓰레드 게시물
- 커뮤니티 글 링크
- 블로그 리뷰 등

### 교차 검증 메커니즘

```text
강남 스시오마카세
├── 유튜브: 성시경 채널 (2024.03)
├── 유튜브: 먹방 유튜버A (2024.01)
├── 커뮤니티: 클리앙 맛집게시판 (2023.12)
├── 인스타그램: @seoul_foodie (2024.02)
└── 맛탐정: @미식가김철수 폴더에 포함
```

**여러 소스가 쌓이면:**
1. 객관성 확보
2. 유튜브 링크로 분위기·서비스·실제 음식 확인
3. 소스가 많을수록 "검증된 맛집"으로 인식

## 5. 맥락 보존 철학

- **미식 아카이브 메타포**: SNS가 아닌 나만의 미식 지식 베이스로 포지셔닝
- **Slack '나중에' 감성**: "지금 저장하고, 나중에 그 근처에 갔을 때 저장 당시 맥락을 복구"하는 도구적 가치
