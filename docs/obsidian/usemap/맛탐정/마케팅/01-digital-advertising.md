# 1. DIGITAL ADVERTISING (15 assets)

## 1.1 Google Ads

### Headlines (30자 이내)

| # | Headline | 글자 수 | 포커스 |
|---|----------|---------|--------|
| H1 | 맛집, 출처부터 확인하세요 | 13 | 핵심 차별점 |
| H2 | 별점 대신 출처로 발견 | 11 | 차별화 |
| H3 | 유튜버 맛집 한곳에서 | 10 | 유튜브 소스 |
| H4 | 광고 리뷰 없는 맛집 앱 | 12 | Pain point |
| H5 | 맛탐정에서 진짜 맛집 발견 | 13 | 브랜드 + 액션 |

**A/B 테스트**: H1(출처 강조) vs H4(Pain point 강조) 우선 테스트. CTR 기준 승자를 H2·H3과 조합.

### Display Ad Concepts

#### 300×250 (Medium Rectangle)

```text
+------------------------------+
|                              |
|  [유튜브 아이콘 + 커뮤니티    |
|   아이콘이 겹치는 이미지]     |
|                              |
|  출처가 다르면,              |
|  발견이 다르다.              |
|                              |
|  [맛탐정 시작하기] (CTA)     |
|                              |
+------------------------------+
```

- **Visual**: Primary 배경(#7C3AED 그라디언트), 출처 배지 3개(유튜브 빨강, 커뮤니티 파랑, 폴더 보라)가 한 장소 카드 위에 겹침
- **Typography**: Tagline은 Headline (Bold), CTA는 Button(white on primary)
- **Color**: Primary gradient → White text

#### 728×90 (Leaderboard)

```text
+------------------------------------------------------------------+
| [출처 배지 3개]  출처가 다르면, 발견이 다르다.  [맛탐정 시작하기]  |
+------------------------------------------------------------------+
```

- **Visual**: 좌측에 출처 아이콘 3개 → 중앙 Tagline → 우측 CTA
- **Color**: White 배경, Primary 텍스트, Primary CTA 버튼

#### 160×600 (Wide Skyscraper)

```text
+------------------+
|                  |
| [음식 이미지]     |
|                  |
| 별점 없이        |
| 출처만으로       |
| 맛집을 발견      |
|                  |
| 유튜브 2         |
| 커뮤니티 1       |
| 폴더 3           |
|                  |
| [발견하기]        |
|                  |
+------------------+
```

- **Visual**: 상단 음식 사진 → 중단 카피 → 하단 출처 배지 예시 → CTA
- **Color**: Surface-50 배경, 출처별 컬러 배지

## 1.2 Facebook/Instagram Ads

### Feed Ad Concepts (3개)

#### Feed Ad 1: "저장만 하고 결국 재검색"

- **Visual**: 네이버 지도 별표가 수백 개 쌓인 스크린샷(블러) → 맛탐정 폴더 UI(깔끔한 정리)로 전환되는 Before/After 이미지
- **Copy**: "맛집 300개 저장해놓고 결국 근처 검색하시는 분?
맛탐정은 출처와 메모를 함께 저장해서, 3개월 뒤에도 '왜 저장했는지' 기억나요."
- **CTA**: 지금 정리 시작하기
- **A/B**: 이미지를 Before/After vs 단일 맛탐정 UI로 테스트

#### Feed Ad 2: "성시경이 갔던 그 국밥집"

- **Visual**: 유튜브 먹방 영상 썸네일 스타일 이미지 → 맛탐정 장소 상세 UI(출처 하이라이트에 "유튜브 2, 커뮤니티 1") 오버레이
- **Copy**: "유튜브에서 봤던 그 국밥집, 어디였더라?
맛탐정이 유튜버 맛집을 다 모아놨어요. 출처별로 검색하고, 폴더에 저장하세요."
- **CTA**: 유튜버 맛집 모아보기
- **A/B**: "성시경" 직접 언급 vs 일반적 "유튜버" 표현

#### Feed Ad 3: "광고 리뷰 말고 진짜 추천"

- **Visual**: 네이버 지도 리뷰(블러/모자이크) 위에 빨간 X 표시 → 맛탐정 교차 검증 UI(출처 4개 연결된 장소 카드)
- **Copy**: "광고인지 진짜인지 구분이 안 되시죠?
맛탐정은 별점 없이, 유튜브·커뮤니티·인스타 출처를 교차 검증해요.
여러 소스가 추천한 곳이면 가볼 만하지 않을까요?"
- **CTA**: 교차 검증된 맛집 보기
- **A/B**: Pain point 강조 vs 솔루션 강조

### Story Ad Concepts (9:16, 3개)

#### Story Ad 1: "3초 Hook — 왜 저장했지?"

- **Frame 1** (0~2s): 검은 화면, 흰 텍스트 "여기 왜 저장했지?" (큰 글씨)
- **Frame 2** (2~4s): 네이버 지도 별표 수백 개 스크린샷
- **Frame 3** (4~6s): 맛탐정 폴더 UI — "유튜브에서 본 국밥집" 메모가 보이는 카드
- **Frame 4** (6~8s): "출처와 메모가 남으니까, 3개월 뒤에도 기억나요" + CTA
- **CTA**: 위로 스와이프 → 앱 다운로드

#### Story Ad 2: "출처 개수 = 신뢰도"

- **Frame 1** (0~2s): "이 맛집, 믿어도 될까?" (큰 텍스트)
- **Frame 2** (2~4s): 출처 배지 1개 → "글쎄..."
- **Frame 3** (4~5s): 출처 배지 3개 → "괜찮을 것 같은데?"
- **Frame 4** (5~7s): 출처 배지 5개 → "가봐야지!" (점점 강조)
- **Frame 5** (7~9s): 맛탐정 로고 + "출처가 다르면, 발견이 다르다" + CTA
- **CTA**: 위로 스와이프

#### Story Ad 3: "폴더 = 나만의 큐레이션"

- **Frame 1** (0~2s): "맛집 저장, 별표 말고 폴더로" (큰 텍스트)
- **Frame 2** (2~5s): 폴더 목록 UI — "데이트 코스", "회식 맛집", "부산 여행"
- **Frame 3** (5~7s): 폴더 내부 UI — 장소 카드 + 메모 + 출처 배지
- **Frame 4** (7~9s): "누구나 맛탐정이 될 수 있어요" + CTA
- **CTA**: 위로 스와이프

### Reel/TikTok Script Concepts (15~30초, 3개)

#### Reel 1: "네이버 별점 4.5의 함정" (20초)

| 시간 | 화면 | 나레이션/자막 |
|------|------|---------------|
| 0~3s | Hook: 네이버 지도 4.5점 음식점 화면 | "네이버 별점 4.5? 근데 후기 보면..." |
| 3~7s | 마케팅 리뷰 스크롤 | "체험단, 협찬, 광고... 진짜 추천이 어딨지?" |
| 7~12s | 맛탐정 앱 화면 전환 | "맛탐정은 별점 없이 출처로 보여줘요" |
| 12~17s | 교차 검증 UI | "유튜브 2개, 커뮤니티 3개가 추천한 곳이면?" |
| 17~20s | 로고 + CTA | "출처가 다르면, 발견이 다르다. 맛탐정" |

#### Reel 2: "유튜브 맛집 검색의 고통" (15초)

| 시간 | 화면 | 나레이션/자막 |
|------|------|---------------|
| 0~3s | 유튜브에서 맛집 영상 보는 중 | "이 맛집 어디지? 영상 다시 찾아봐야 하나..." |
| 3~8s | 맛탐정에서 "성시경" 검색 → 장소 리스트 | "맛탐정이 유튜버별로 맛집 다 정리해놨어요" |
| 8~12s | 장소 카드 탭 → 상세 + 유튜브 출처 링크 | "출처 영상도 바로 확인 가능" |
| 12~15s | 로고 | "맛탐정에서 유튜버 맛집 검색하세요" |

#### Reel 3: "맛집 폴더 정리 ASMR" (25초)

| 시간 | 화면 | 나레이션/자막 |
|------|------|---------------|
| 0~5s | 맛탐정 앱에서 폴더 만들기 | (ASMR 타이핑 소리) "데이트 코스" 폴더 생성 |
| 5~10s | 장소 검색 → 카드에서 저장 버튼 | 장소를 하나씩 폴더에 저장하는 과정 |
| 10~17s | 폴더 내 장소 리스트 완성 | 메모 작성: "유튜브에서 본 파스타 맛집" |
| 17~22s | 친구에게 폴더 공유 | "초대 코드 복사" → 카톡으로 공유 |
| 22~25s | 로고 | "나만의 맛집 아카이브, 맛탐정" |

## A/B 테스트 종합

| 테스트 | 변수 A | 변수 B | 측정 |
|--------|--------|--------|------|
| Google Headline | H1 출처 강조 | H4 Pain point | CTR |
| Google Description | D2 시나리오형 | D1 기능형 | CTR |
| Feed Ad | Before/After 이미지 | 단일 UI | CTR, 설치 |
| Story Ad | 3초 텍스트 Hook | 3초 영상 Hook | 완료율 |
| Reel | 네이버 Pain point 시작 | 유튜브 시나리오 시작 | 조회수, 공유 |
