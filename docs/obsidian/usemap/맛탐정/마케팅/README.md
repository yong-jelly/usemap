# 맛탐정 마케팅 에셋 라이브러리

모바일 앱 런칭·인지도 확보를 위한 마케팅 자산 모음.

## Campaign Brief

| 항목 | 내용 |
|------|------|
| 제품 | 맛탐정 — 출처 기반 맛집 발견 플랫폼 |
| 목표 | AWARENESS (인지도 확보) → CONVERSION (가입·저장) |
| 타겟 | 20~40대, 맛집 정보 활발히 소비, 광고 리뷰 회의적, 유튜브·인스타 맛집 콘텐츠 소비자 |
| 테마 | "출처가 다르면, 발견이 다르다" |
| 톤 | MINIMAL + PROFESSIONAL (신뢰감 우선) |

## Messaging Hierarchy

| 계층 | 메시지 |
|------|--------|
| Tagline | 출처가 다르면, 발견이 다르다 |
| Value Prop | 유튜브·커뮤니티·인스타 출처를 교차 검증하여 신뢰할 수 있는 맛집을 발견 |
| Proof Point | 별점 없이, 출처 정보만으로 맛집을 판단하는 새로운 방식 |
| CTA | 지금 맛탐정에서 발견하기 |
