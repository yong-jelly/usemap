# 3. PATTERNS

## 3.1 Page Templates

### Landing (온보딩)

```text
+----------------------------------+
|        [로고/브랜드]              |
|   흩어진 맛집 정보의 허브         |
|   출처 기반 발견                  |
+----------------------------------+
|  [시작하기]  [로그인]             |
+----------------------------------+
```

- 히어로: 한 줄 정의, 핵심 가치
- CTA: 시작하기(primary), 로그인(ghost)

---

### Dashboard (홈)

```text
+----------------------------------+
| [위치] [검색]                    |
+----------------------------------+
| [추천] [소스] [구독]  (탭)       |
+----------------------------------+
| [PlaceCard] [PlaceCard] ...      |
| [PlaceCard] [PlaceCard] ...      |
+----------------------------------+
| [홈] [피드] [맛탐정] [프로필]    |
+----------------------------------+
```

- 상단: 위치 설정, 검색 진입
- 탭: 추천 / 소스 탐색 / 구독 피드
- 리스트: PlaceCard, 무한 스크롤

---

### Settings (설정)

```text
+----------------------------------+
| < 설정                           |
+----------------------------------+
| 프로필 수정                       |
| 알림                             |
| 테마                             |
| 로그아웃                         |
+----------------------------------+
```

- List 패턴, Chevron으로 이동

---

### Profile (프로필)

```text
+----------------------------------+
| [Avatar] [닉네임] [편집]         |
| [구독 N] [구독자 N]              |
+----------------------------------+
| [최근] [좋아요] [콜렉션] [저장]  |
| [방문] [리뷰] [구독] [구독자]    |
| [분석]                           |
+----------------------------------+
| [FolderCard] [FolderCard] ...     |
+----------------------------------+
```

- 상단: 프로필 정보, Stats
- 그리드: 메뉴 칩 → 해당 리스트

---

### Checkout (결제)

- 맛탐정은 유료 결제 없음. 대신 **폴더 초대 수락** 플로우에 적용 가능.
- 초대 코드 입력 → 폴더 참여 확인 → 완료

---

## 3.2 User Flows

### Onboarding

1. 스플래시 → 로그인/회원가입
2. (선택) 위치 권한 요청
3. 홈 진입
4. (선택) 첫 폴더 만들기 가이드

---

### Authentication

1. 로그인 필요 시 → 모달/시트
2. OAuth (카카오, 구글 등)
3. 이메일 등록 (회원가입)
4. 완료 → 이전 화면 복귀

---

### Search

1. 검색 입력 포커스
2. 최근 검색어 표시 (있으면)
3. 입력 시 실시간 제안 (선택)
4. 결과: PlaceCard 리스트
5. 필터: 지역, 카테고리, 테마

---

### Filtering

1. 필터 버튼/시트 열기
2. 지역 칩, 카테고리 칩, 테마 칩
3. 적용 → 결과 갱신
4. 초기화 버튼

---

### Empty States

| 상황 | 메시지 | CTA |
|------|--------|-----|
| 검색 결과 없음 | "조건에 맞는 장소가 없어요" | 필터 초기화 |
| 폴더 비어있음 | "아직 저장한 장소가 없어요" | 탐색하기 |
| 구독 피드 없음 | "구독한 소스가 없어요" | 소스 탐색 |
| 좋아요 없음 | "좋아요한 장소가 없어요" | 탐색하기 |

---

## 3.3 Feedback Patterns

### Success

- Toast: "폴더에 저장했어요", "복사했어요"
- 아이콘: check, primary
- 2~3초 후 자동 닫힘

---

### Error

- Alert: 오류 메시지, 재시도 버튼
- Toast: 네트워크 오류 등 간단한 경우

---

### Loading

- Spinner: 버튼 내, 전체 화면
- Skeleton: 리스트, 카드
- Pull-to-refresh: 피드

---

### Empty

- 일러스트/아이콘 (선택)
- 메시지: 명확한 설명
- CTA: 다음 액션 유도
