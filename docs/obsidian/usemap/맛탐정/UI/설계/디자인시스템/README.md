# 맛탐정 디자인 시스템

모바일 앱 UI/UX 재설계를 위한 디자인 시스템.
Apple HIG 원칙 기반. 현재 기능을 참고하여 재개발 시 적용할 수 있도록 작성함.

## 브랜드 속성

| 항목 | 값 |
|------|-----|
| Personality | MINIMALIST + TRUST |
| Primary emotion | TRUST (출처 기반 신뢰), CALM (막힘 없는 탐색) |
| Target audience | 20~40대, 맛집 수집·관리 니즈, 광고 리뷰 회의적 |

## 핵심 원칙 (맛탐정)

1. **Actionable**: 모든 정보는 "나중에 방문할 때" 도움이 되어야 함
2. **Traceable**: 모든 정보는 "어디서 왔는지" 추적 가능해야 함
3. **Personal**: 나의 메모와 나의 맥락이 우선
