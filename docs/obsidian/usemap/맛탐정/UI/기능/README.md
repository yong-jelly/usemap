# 맛탐정 UI 기능 문서

UI 내 모든 사용자 행동(~하기)을 나열하고 상세 정의함.
UI/UX 설계 개선 목적. 모델링·함수 문서와 동일한 드라이한 스타일.

## 기능 정의 형식

각 기능은 다음 요소로 묘사함:
- **위치**: 화면/컴포넌트
- **트리거**: 사용자 행동(클릭, 입력 등)
- **결과**: 발생하는 변화
- **전제조건**: 로그인 등
- **관련 API**: RPC/엔드포인트
