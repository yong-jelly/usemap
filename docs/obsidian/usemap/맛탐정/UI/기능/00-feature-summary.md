# UI 기능 요약

사용자가 수행할 수 있는 모든 UI 행동(~하기) 목록.

## 기능 목록 (표)

| 기능 | 위치 | 한 줄 정의 |
|------|------|-------------|
| 좋아요하기 | 장소 카드, 장소 상세 | 장소에 좋아요 토글. 관심 표시 |
| 저장하기 | 장소 카드, 장소 상세 | 장소를 폴더에 저장. 콜렉션 저장 시트 |
| 폴더에 저장하기 | 폴더 선택 모달 | 장소를 선택한 폴더에 추가/제거 |
| 리뷰 쓰기 | 장소 상세, 액션 시트 | 장소에 사용자 리뷰 작성 |
| 다녀왔어요 | 장소 상세, 액션 시트 | 방문 기록 추가·수정·삭제 |
| 관련 콘텐츠 추가하기 | 장소 상세, 액션 시트 | 유튜브/커뮤니티 링크 등록 |
| 댓글하기 | 장소 상세 | 장소에 댓글 작성·수정·삭제·좋아요 |
| 네이버 지도 보기 | 장소 상세, PlaceActionRow | 네이버 지도 앱/웹으로 이동 |
| 출처 보기 | 장소 상세 | 유튜브/폴더/커뮤니티 출처 링크 |
| 구독하기 | 소스 상세, 폴더 카드 | 폴더·채널 구독 토글 |
| 공유하기 | 소스 상세 | URL 클립보드 복사 |
| 폴더 만들기 | 폴더 생성 페이지, 모달 | 새 맛탐정 폴더 생성 |
| 폴더 수정하기 | 폴더 설정 시트 | 제목·설명·권한 수정 |
| 폴더 삭제하기 | 소스 상세(관리자) | 폴더 삭제 |
| 폴더에서 장소 제거하기 | 폴더 상세 | 장소 제거·메모 수정 |
| 폴더 초대하기 | 폴더 생성 후 | 초대 코드 복사·공유 |
| 초대 코드 입력하기 | - | 폴더 초대 수락 |
| 검색하기 | 검색 페이지, 탐색 페이지 | 지역·음식점 검색 |
| 필터 적용하기 | 검색, 탐색 | 지역·카테고리·테마별 필터 |
| 정렬하기 | 홈, 피드 | 최신순·거리순 |
| 위치 설정하기 | 위치 시트 | 현재 위치 저장·선택 |
| 로그인하기 | 인증 모달 | OAuth 로그인 |
| 회원가입하기 | 인증 모달 | 이메일 등록 |
| 프로필 수정하기 | 프로필 편집 | 닉네임·소개·이미지 |
| 탭 전환하기 | 홈, 피드, 맛탐정 | 추천/소스/구독 등 |
| 장소 상세 보기 | 카드 클릭 | 장소 상세 모달/페이지 |
| 지도 보기 | 소스 상세 | 지도 페이지로 이동 |
| 리스트 수정하기 | 폴더 옵션 시트 | 폴더 설정 시트 열기 |
| 폴더에 저장 | PlaceCard | 저장 버튼 토글 후 컬렉션 시트 |
| 최근 검색어 삭제하기 | 검색 | 검색 이력 전체 삭제 |
| 검색 종료하기 | 탐색 | 검색 모드 종료 |
| 내 리뷰 필터 | 리뷰 모달 | 내 리뷰만 보기 |
| 방문 기록 추가·수정·삭제 | 방문 모달 | 방문일·동행·메모 |
| 메모 작성·수정 | 폴더 내 장소 | 폴더별 장소 메모 |
| 폴더에서 제거 | PlaceCommentForm | 장소 제거 확인 |
| 지역 선택하기 | 탐색, 검색 | group1·group2 칩 |
| 조건 초기화 | 검색, 탐색 | 필터 초기화 |
| 탐색하기 | 홈 | 맛탐정 탭으로 이동 |
| 홈/피드/맛탐정 이동 | 하단 네비 | 전역 네비게이션 |
| 프로필 메뉴 이동 | 프로필 | 최근/좋아요/콜렉션/저장/방문/리뷰/구독/구독자/분석 |
| 테마·지역 선택 | 랭킹 | 랭킹 필터 |
| 인스타 콘텐츠 저장 | 인스타 파서 | 벌크 저장 |
| 인스타 장소 매핑 | 인스타 파서 | 콘텐츠-장소 연결 |

## 기능별 화면 매핑

| 화면 | 주요 기능 |
|------|-----------|
| 장소 상세 모달 | 좋아요, 저장, 댓글, 네이버, 리뷰쓰기, 다녀왔어요, 관련콘텐츠, 출처 |
| PlaceCard | 좋아요, 저장, 장소 상세, 폴더 저장 |
| 홈 | 탭전환, 위치설정, 소스탐색, 구독 |
| 피드 | 정렬, 필터, 장소 상세 |
| 검색 | 검색, 필터, 최근검색어삭제 |
| 탐색 | 검색, 필터, 지역선택, 검색종료 |
| 맛탐정 | 소스탐색, 구독 |
| 소스 상세 | 구독, 공유, 지도, 삭제(관리자), 설정(폴더) |
| 폴더 상세 | 장소 제거, 메모, 구독 |
| 프로필 | 메뉴이동, 편집 |
| 인증 모달 | 로그인, 회원가입, 닫기 |
