# 인증·네비게이션 관련 UI 기능

로그인, 회원가입, 전역 네비게이션.

## 로그인하기

| 항목 | 내용 |
|------|------|
| **위치** | AuthModal (useAuthModalStore.openLogin) |
| **트리거** | OAuth 버튼(구글 등) 클릭 |
| **결과** | Supabase Auth 리다이렉트. /auth/callback 복귀 |
| **UX** | "로그인으로 돌아가기" (회원가입 뷰에서). ESC·배경 클릭으로 닫기 |

## 회원가입하기

| 항목 | 내용 |
|------|------|
| **위치** | AuthModal (view=signup) |
| **트리거** | 로그인 모달에서 "회원가입" 링크. 이메일·비밀번호 입력 후 가입 |
| **결과** | auth.users 생성. handle_new_user 트리거로 tbl_user_profile 자동 생성 |
| **UX** | 단계별. 이메일 인증 등 |

## 로그인 모달 닫기

| 항목 | 내용 |
|------|------|
| **위치** | AuthModal 헤더 |
| **트리거** | ChevronLeft(뒤로), X(닫기) |
| **결과** | useAuthModalStore.close |

## 홈/피드/맛탐정 이동 (하단 네비)

| 항목 | 내용 |
|------|------|
| **위치** | BottomNav |
| **트리거** | 홈(관리자만), 피드, 맛탐정 아이콘 클릭 |
| **결과** | /home, /feed, /feature 이동. 스크롤 상단 복구 |
| **UX** | 프로필(/profile) 비로그인 시 openLogin. 스크롤 시 네비 숨김 |

## 스크롤 시 네비 숨김

| 항목 | 내용 |
|------|------|
| **위치** | BottomNav |
| **정의** | 스크롤 다운 시 y:60으로 슬라이드 아웃. 스크롤 업 또는 200ms 후 복귀 |

## 사이드바

| 항목 | 내용 |
|------|------|
| **위치** | Sidebar |
| **정의** | 햄버거 메뉴 등. 프로필·설정 등 링크 |
